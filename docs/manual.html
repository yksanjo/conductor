🎼 Conductor Cockpit

quick manual · github.com/yksanjo/conductor
Open it
conductor up → opens the cockpit at localhost:7591. Or conductor for a quick terminal table.
Status dots
working now open (idle) recent idle
Reply to one
Type in a card's box and press ↵, or tap a quick button:
YesNoContinueReviewRe-iterateTest+deploy
Prompt all
Use the ⚡ Prompt all managed bar at the top to message every managed window at once.
New window
+ New window button (name + folder), or conductor run <name>. It launches in tmux and auto-accepts the trust prompt.
Adopt existing
Reply on any plain window's card to pull it under control, or conductor adopt <id>. Then close the original tab.
Type a lot
conductor attach <name> drops you into the window to type long commands directly.
From any agent
It's an MCP server too: "use conductor to list my sessions / what's left?"
How control works: a reply goes to the window's Claude session running in tmux — watch the response in the card's "doing now" line, or conductor attach to see it live. Plain terminal windows can't take input directly, so replying to one adopts it (forks it into a managed copy) first.
